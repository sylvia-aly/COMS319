To compile and run task 1:
Double click on 'snake.HTML'
Push start button to run the snake and stop to stop button the snake
If the snake touches its own body it will stop the snake, you have to refresh the page to start over
If the snake go out of bounds it will stop the snake, you have to refresh page to start over
Press 'Turn Left' button to turn the snake to the Left
Press 'Turn Right' button to turn the snake to the Right
